Under the Hood

This Todo application is built with Lite-SPA, a minimal Vanilla JS SPA template. By combining standard browser APIs with minimal libraries, it provides a robust single page application experience without complex build tools.

Reactivity: Powered by @preact/signals-core. When you add, check, or delete a task, the DOM reacts and updates only the necessary nodes.

Routing: Handled by page.js. The URL paths change without page reloads, and views are swapped dynamically via class toggling.

Persistence: Enabled by browser localStorage. Data is automatically written to local storage using signal side-effects (effect).
